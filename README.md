### Semana-Omnistack10

```
 - Projeto relacionado a Semana Omnistack(10) proposto pela empresa RocketSeat;
 - Back-End: NodeJs + Mongoose + MongoDb;
 - Front-End: ReactJs;
 - Proposta: Cadastro e busca de desenvolvedores e suas tecnologias com integração a api do github;
```

#### Features:
```
  - Listagem de Desenvolvedores (integrado com a api do github);
  - Cadastro de Desenvolvedor (utilização de latitude e longitude);
  - Deleção de um Desenvolvedor Específico;
```

#### Como Iniciar?

#### 1 - Clonando o projeto
```
 - Clonar o projeto via Git;
 - git clone -url-do-projeto
```
#### 2 - Back-End
```
 - cd Semana-Omnistack10/backend
 - yarn dev
```

#### 3 - Front-End  
```
 - cd Semana-Omnistack10/web
 - yarn start
```
